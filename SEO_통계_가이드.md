# 홈페이지 통계 및 검색엔진(SEO) 등록 가이드

본 문서는 홈페이지 접속 통계를 확인하고, 구글과 네이버 검색에 홈페이지가 잘 노출되도록 사이트 소유권을 등록하는 절차를 안내합니다.

## 1. 홈페이지 접속 통계 확인하기

홈페이지 접속 통계(방문자 수 등)를 확인하는 방법에는 두 가지가 있습니다.

### 방법 A: GitHub 기본 통계 (별도 설치 없음, 추천도: 낮음)
- **확인 방법**: [GitHub 저장소 페이지](https://github.com/bae4127-hub/homepage) 접속 → 상단의 **Insights** 탭 클릭 → 좌측 메뉴에서 **Traffic** 클릭
- **특징**: 아무 설정이 필요 없으나, 최근 14일 치의 기초적인 조회수와 방문자 수만 제공됩니다.

### 방법 B: 전문 통계 분석 도구 연동 (추천도: 매우 높음)
구글 애널리틱스나 네이버 애널리틱스(무료)를 홈페이지에 연결하는 방법입니다.
방문자의 유입 경로(검색어, 페이스북 등), 기기(스마트폰/PC), 접속 시간대 등을 영구적으로 상세히 알 수 있습니다. 네이버 블로그에 익숙하시다면 한국어 메뉴가 직관적인 '네이버 애널리틱스'를 추천합니다.

**연결 방법**:
1. [구글 애널리틱스](https://analytics.google.com/) 또는 [네이버 애널리틱스](https://analytics.naver.com/) 홈페이지에 접속하여 로그인합니다.
2. 사이트 주소(`https://cnksch.or.kr`)를 등록합니다.
3. 등록 후 발급되는 **'추적 코드(알파벳과 숫자로 된 짧은 코드)'**를 복사해 둡니다.
4. 해당 코드를 Claude에게 알려주시면, 에이전트(Claude)가 `index.html` 파일에 해당 코드를 삽입해 드립니다.

## 2. 검색엔진 최적화(SEO) 사이트 소유권 등록 (필수)

홈페이지 뼈대 안에는 검색엔진 노출을 위한 모든 세팅(`sitemap.xml`, `robots.txt`, 메타 태그 등)이 이미 완료되어 있습니다. 마지막으로 아래 두 곳에 접속하셔서 **사이트 소유권 등록**을 진행해 주셔야 합니다.

### 2-1. 네이버 서치어드바이저 (네이버 검색 노출용)
1. [네이버 서치어드바이저](https://searchadvisor.naver.com/)에 접속하여 네이버 아이디로 로그인합니다.
2. 우측 상단 **[웹마스터 도구]** 버튼을 클릭합니다.
3. 사이트 등록 칸에 `https://cnksch.or.kr` 을 입력합니다.
4. **'사이트 소유확인'** 단계에서 **'HTML 태그'** 방식을 선택합니다.
5. 화면에 나타나는 초록색 글씨의 `<meta name="naver-site-verification" content="어쩌고저쩌고..."/>` 코드를 복사해서 Claude에게 알려주세요.
6. Claude가 뼈대 파일(`index.html`)에 코드를 삽입했다고 안내해 드리면, 그때 네이버 화면에서 **[소유확인]** 버튼을 누르시면 됩니다.
7. 완료 후 좌측 메뉴의 '요청' -> '사이트맵 제출' 메뉴에서 `https://cnksch.or.kr/sitemap.xml` 을 제출합니다.

### 2-2. 구글 서치 콘솔 (구글 검색 노출용)
1. [구글 서치 콘솔](https://search.google.com/search-console/about)에 접속하여 구글 아이디(교회 계정 권장)로 로그인합니다.
2. 속성 유형 선택 화면에서 오른쪽의 **[URL 접두어]** 방식을 선택하고 `https://cnksch.or.kr` 을 입력합니다.
3. 소유권 확인 화면이 나오면, 스크롤을 내려 **[HTML 태그]** 방식을 선택합니다.
4. `<meta name="google-site-verification" content="어쩌고저쩌고..."/>` 코드를 복사해서 Claude에게 알려주세요.
5. Claude가 코드를 삽입한 뒤, 구글 화면에서 **[확인]** 버튼을 누르시면 완료됩니다.
6. 완료 후 좌측 메뉴의 'Sitemaps(사이트맵)' 메뉴에서 `https://cnksch.or.kr/sitemap.xml` 을 제출합니다.
