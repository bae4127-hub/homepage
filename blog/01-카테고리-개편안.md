# 블로그 카테고리 개편안 (세움교회 모델 적용, 2026-06-12 갱신)

제주 세움교회 블로그의 성공 요소 3가지를 우리 것으로 옮기는 개편안:
1. **홈페이지형 첫인상** (배너+투명위젯+프롤로그) → `08-홈페이지형-배너-위젯-가이드.md`
2. **일관된 카테고리 네이밍** (세움: "○○로 세우다") → 우리: "○○로 피우다" (비전 "복음을 꽃 피우는 청라강성교회"와 연결)
3. **매주 빠짐없는 발행 루틴** → 우리는 이미 오렌지카드·선포기도문·주보를 매주 만들고 있으므로 "블로그에 옮기기"만 하면 됨

블로그 관리 → 메뉴·글·동영상 관리 → 블로그(카테고리 관리)에서 변경.

## 새 카테고리 구조 (추천안 — "피우다" 네이밍)

```
📌 처음 오신 분                      ← 신설 (글 1개, 공지 고정)
🏠 청라강성교회                      ← 신설 (소개)
   ├ 인사말과 비전
   ├ 예배시간 안내
   └ 오시는 길
🎬 말씀으로 피우다                   ← 예배와 말씀
   ├ 주일예배                        (기존 '설교' 이름 변경, 재가동)
   ├ 말씀쇼츠                        (기존 유지 41글, 재개)
   └ 공과강의                        (기존 유지 25글)
🍊 가정으로 피우다                   ← 신설 (홈페이지 '가정예배·밤기도회' 코너와 대응)
   ├ 오렌지카드(가정예배)
   └ 선포기도문(밤기도회)
📰 소식으로 피우다                   ← 교회소식
   ├ 주보·공지                       (신설, 매주)
   └ 사역이야기                      (기존 유지 51글)
💚 사랑으로 피우다                   ← 이웃사랑 (기존 '분별력 있는 순종' 중 나눔 글 이동)
✍ 목회칼럼                          ← '일일묵상'(108) + '나의 글쓰기'(21) 통합
```

> 네이밍이 부담스러우면 평범한 명칭(예배와 말씀 / 가정예배·밤기도회 / 교회소식 / 이웃사랑)으로 해도
> 구조는 동일. 단, 세움교회 사례에서 통일 네이밍은 "기억에 남는 교회"를 만드는 비용 0의 브랜딩임.

## 기존 카테고리 처리표

| 기존 | 글 수 | 처리 |
|---|---|---|
| 설교 | 20 | '주일예배'로 이름 변경, 재가동 |
| 말씀쇼츠 | 41 | 유지 (말씀으로 피우다 아래로) |
| 공과강의 | 25 | 유지 (말씀으로 피우다 아래로) |
| 사역이야기 | 51 | 유지 (소식으로 피우다 아래로) |
| 이웃사랑>분별력 있는 순종 | 16 | 나눔 활동 글만 '사랑으로 피우다'로, 칼럼성 글은 '목회칼럼'으로 |
| 일일묵상 | 108 | '목회칼럼'으로 이동 후 카테고리 삭제 |
| 나의 글쓰기 | 21 | '목회칼럼'으로 이동 후 카테고리 삭제 |
| 홍보영상 | 3 | **비공개** (이전 사역지 콘텐츠) |
| 낙서장 / 책읽기 / 일상기록 / 포토 관련 | ~60 | 비공개 |

> 비공개 = 카테고리 공개설정만 끄면 됨. 글은 지워지지 않음.

| 콘텐츠 | 원본 위치 | 블로그 작업 | 템플릿 |
|---|---|---|---|
| 주일예배 영상+요약 | 유튜브(자동 업로드 중) | 임베드+요약 15분 | 06 |
| 주보 | `2026 01 주보\` (매주 hwpx) | 텍스트 복사 10분 | 07 |
| 오렌지카드 | `2026 03 오렌지카드\` (매주 PNG) | 이미지+텍스트 10분 | 09 (완성본: 11) |
| 선포기도문 | `2026 08 선포기도문\` (매주 hwpx/PDF) | 텍스트 복사 5분 | 10 (완성본: 12) |

이미 매주 만들고 있는 4종을 옮기는 것이므로 주당 총 40분 안팎.

## 과거 아카이브 소급 게시 전략

- 오렌지카드·선포기도문 24주치, 주보 23주치 보유
- 한꺼번에 올리면 네이버가 도배성으로 판단할 위험 → **이번 주 것부터 시작, 과거분은 주 2~3개씩** 역순으로
- 약 2~3개월이면 아카이브 전체가 블로그 검색 자산으로 전환됨

## 글 제목 규칙 (검색 노출용)

- 주일예배: `[주일예배] 설교제목 | 성경본문 | 2026.6.14 | 청라 강성교회`
- 주보: `[주보] 2026년 6월 14일 주일 | 교회소식 | 청라 강성교회`
- 오렌지카드: `[오렌지카드] 주제 | 본문 | 날짜 | 청라강성교회 가정예배`
- 선포기도문: `[선포기도문] 제목 | 본문 | 날짜 | 청라강성교회 밤기도회`
- 공통: 제목에 **"청라"** 포함(지역 검색), 주일설교와 같은 본문은 같은 주제어 사용(한 본문 검색에 3~4글 노출)
- ⚠️ "[오렌지카드]"/"[선포기도문]"은 홈페이지 자동 표시 트리거이므로 표기 유지
