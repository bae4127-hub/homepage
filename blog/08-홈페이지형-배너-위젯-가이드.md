# 홈페이지형 블로그 만들기 (세움교회 방식)

배너 이미지 + 투명 위젯 + 프롤로그형 메인, 세 가지 조합입니다. 코딩이 아니라 "설정"입니다.
⚠️ 이 레이아웃은 **PC에서만** 보입니다. 모바일은 일반 블로그로 표시되므로
새가족안내 글 공지 고정(02번 글)이 모바일 대응입니다.

---

## 1단계. 배너(타이틀) 이미지 제작 — 캔바

**규격: 가로 966px × 세로 400px** (네이버 타이틀 최대 가로 966, 세로 50~600)

배너에 들어갈 내용:
```
┌──────────────────────────────────────────────┐
│  [교회 전경 또는 청라 풍경 사진 배경]                  │
│                                              │
│   청라강성교회                                   │
│   [비전 한 문장 — 예: 청라에 복음을 꽃 피우다]         │
│   주일예배 [11:00] | 인천 청라 6단지                 │
│                                              │
│ ──────────── 하단 40px 메뉴 띠 ────────────────  │
│  새가족안내 │ 교회소개 │ 예배와말씀 │ 교회소식 │ 오시는길 │
└──────────────────────────────────────────────┘
```
- 메뉴 글자 5개를 **하단 40px 띠**에 균등 간격으로 배치 (이 위에 투명 위젯을 겹칩니다)
- 글자는 흰색+반투명 어두운 띠 배경이면 어떤 사진에도 잘 보임

등록: 블로그 관리 → 꾸미기 설정 → 세부 디자인 설정(리모콘) → 타이틀 → 직접등록 → 파일 업로드

## 2단계. 레이아웃 설정

블로그 관리 → 꾸미기 설정 → 레이아웃·위젯 설정
- 레이아웃: **1단 레이아웃** 선택 (위젯이 가로로 나란히 놓이는 형태)
- 타이틀 바로 아래에 위젯 5개가 한 줄로 오도록 배치

## 3단계. 투명 위젯 5개 등록 (메뉴 버튼화)

레이아웃·위젯 설정 → 위젯 직접등록 → 아래 코드를 위젯 1개당 1개씩, 총 5개 등록.
`categoryNo=` 뒤의 번호는 개편 후 **각 카테고리 주소에서 확인** (카테고리 클릭 → 주소창의 categoryNo 값).

```html
<!-- 위젯 1: 새가족안내 -->
<div style="width:170px; height:40px; overflow:hidden; margin-top:-45px;">
<a href="https://blog.naver.com/PostList.naver?blogId=wormwood79&categoryNo=[새가족카테고리번호]&from=postList"
   title="새가족 안내" style="display:block; width:170px; height:40px;"></a>
</div>
```

```html
<!-- 위젯 2: 교회소개 -->
<div style="width:170px; height:40px; overflow:hidden; margin-top:-45px;">
<a href="https://blog.naver.com/PostList.naver?blogId=wormwood79&categoryNo=[교회소개번호]&from=postList"
   title="교회소개" style="display:block; width:170px; height:40px;"></a>
</div>
```

```html
<!-- 위젯 3: 예배와 말씀 -->
<div style="width:170px; height:40px; overflow:hidden; margin-top:-45px;">
<a href="https://blog.naver.com/PostList.naver?blogId=wormwood79&categoryNo=[예배와말씀번호]&from=postList"
   title="예배와 말씀" style="display:block; width:170px; height:40px;"></a>
</div>
```

```html
<!-- 위젯 4: 교회소식 -->
<div style="width:170px; height:40px; overflow:hidden; margin-top:-45px;">
<a href="https://blog.naver.com/PostList.naver?blogId=wormwood79&categoryNo=[교회소식번호]&from=postList"
   title="교회소식" style="display:block; width:170px; height:40px;"></a>
</div>
```

```html
<!-- 위젯 5: 오시는 길 (글 하나로 바로 연결) -->
<div style="width:170px; height:40px; overflow:hidden; margin-top:-45px;">
<a href="[오시는길 글 주소 — 게시 후 복사]"
   title="오시는 길" style="display:block; width:170px; height:40px;"></a>
</div>
```

**미세 조정**: 위젯 줄과 배너 메뉴 글자가 어긋나면
- `margin-top:-45px` 값을 ±5px씩 조정 (위로 더 올리려면 숫자를 키움)
- 배너의 메뉴 글자 간격을 위젯 폭(170px) 기준으로 다시 맞춤
- PC 화면에서 마우스를 올려 커서가 손가락 모양으로 바뀌는 위치 확인

## 4단계. 프롤로그형 메인 설정

블로그 관리 → 메뉴·글·동영상 관리 → 상단메뉴 설정
- 대표메뉴: **프롤로그** 선택
- 프롤로그 관리에서 보기 형식: **이미지 강조형(썸네일 격자)**
- 메인에 보여줄 카테고리: 주일예배, 사역이야기, 말씀쇼츠 추천

## 5단계. 모바일 보완 (중요 — 방문자 다수가 모바일)

- [ ] 새가족안내 글 → 공지로 고정 (모바일 최상단 노출)
- [ ] 블로그 관리 → 기본 설정 → 블로그 정보 → 소개글에:
      `인천 청라신도시 6단지 | 주일예배 [시간] | 처음 오신 분 환영합니다`
- [ ] 프로필 이미지를 교회 로고로 교체
