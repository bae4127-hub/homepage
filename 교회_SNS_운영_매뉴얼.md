# 청라강성교회 공식 SNS 운영 매뉴얼

이 문서는 교회의 페이스북, 인스타그램 및 네이버 블로그 자동화 연동 시스템에 대한 운영 가이드입니다.

## 1. 운영 구조 요약
* **원천 소스(글 작성):** 네이버 블로그
* **자동 배포 (블로그 글):** 목사님이나 담당자께서 네이버 블로그에 새 글을 쓰시면, `dlvr.it` 서비스를 통해 **페이스북 페이지**로 알아서 자동 게시됩니다. (우리가 구축 중인 교회 홈페이지에도 이와 동시에 업데이트 됩니다.)
* **동시 업로드 (쇼츠 영상 등):** 유튜브 쇼츠용 영상이나 중요한 행사 포스터 등은 **메타 비즈니스 스위트(Meta Business Suite)**를 사용하여 페이스북과 인스타그램에 한 번에 업로드합니다.

---

## 2. 자동 게시 시스템 (dlvr.it) 관리
네이버 블로그의 글을 페이스북 페이지로 배달해 주는 무료 자동화 서비스입니다. 설정이 한 번 끝나면 평소에는 사이트에 들어갈 필요가 없습니다.

* **접속처:** [dlvr.it](https://dlvrit.com/) (가입하신 계정으로 로그인)
* **핵심 설정 체크리스트:**
  * 페이스북에 썸네일 사진이 크게 잘 나오도록 하려면 설정의 `Post as`를 **`Photo`**로 유지해야 합니다.
  * **`Post URL`**이 반드시 **ON(파란색)** 상태여야 페이스북 게시물에 블로그 주소가 나타납니다.
  * **`Post Body`**는 끄거나 제한(예: 200자)하는 것이 화면을 깔끔하게 유지하는 비결입니다.
* **참고사항:** 네이버 블로그에 글을 '전체공개'로 발행한 뒤, 페이스북에 등록되기까지 보통 5분~15분 정도의 시차가 발생할 수 있습니다.

---

## 3. 메타 비즈니스 스위트 (페이스북+인스타그램 동시 관리)
새로 만드신 '인스타그램 프로페셔널 계정'과 '페이스북 페이지'가 완벽히 연동되었으므로 사용하는 공식 관리 도구입니다.

* **접속처:** PC에서는 [Meta Business Suite](https://business.facebook.com/) 사이트 접속 / 스마트폰에서는 **`Business Suite`** 앱 설치 후 페이스북 아이디로 로그인
* **활용 방법 (유튜브 쇼츠 영상을 인스타 릴스로 올리기):**
  1. 이미 만들어두신 유튜브 쇼츠 영상 파일(.mp4)을 폰이나 PC에 준비합니다.
  2. 비즈니스 스위트 메인 화면에서 **[릴스 만들기]** 버튼을 클릭합니다.
  3. 상단 설정에서 **페이스북과 인스타그램 아이콘이 모두 체크(선택)**되어 있는지 확인합니다.
  4. 영상 파일을 첨부하고 짧은 문구를 적은 뒤 **[공유]** (혹은 원하는 날짜로 **[예약]**) 버튼을 누릅니다.
  5. 단 한 번의 업로드로 두 플랫폼에 동일한 영상이 동시 배포됩니다!

---

## 4. 추후 관리자 인수인계 방법
* **페이스북 페이지:** 담당자가 바뀔 경우, 기존 관리자가 페이스북 페이지 설정에 들어가서 '새로운 담당자의 개인 페이스북 계정'을 검색해 관리자 권한을 부여해 주면 됩니다. 절대 개인 계정 비밀번호를 통째로 넘겨줄 필요가 없습니다.
* **인스타그램:** 인스타그램은 교회용으로 완전히 분리하여 생성하셨으므로, 가입하실 때 쓰신 `아이디`와 `비밀번호`만 안전하게 기록해 두셨다가 다음 담당자에게 인계하시면 됩니다.
