블로그·유튜브와 자동 연동되는 교회 홈페이지 샘플 모음.
`[대괄호]`로 표시된 부분만 나중에 내용을 채우면 됩니다.

## ⚠️ 홈페이지 미리보기 방법

`index.html`을 **직접 더블클릭하면 안 됩니다** — 브라우저 보안 정책(file://) 때문에
유튜브 영상과 소식 카드가 차단되어 빈 화면처럼 보입니다.

대신 **`site\미리보기.bat`을 더블클릭**하세요.
검은 창과 함께 브라우저가 자동으로 열리고 모든 것이 정상 표시됩니다.
(검은 창을 닫으면 미리보기가 종료됩니다. 인터넷에 배포한 뒤에는 이런 과정 없이 누구나 바로 봅니다.)

## 행사 고정 배너 관리법 (여름성경학교 등)

홈페이지 메인 상단에 고정 배너가 표시됩니다. **`site\data\notice.json` 파일 하나만 수정**하면 됩니다.
여러 개를 동시에 띄울 수 있습니다 — 목록에 `{ }` 항목을 추가하면 되고, 2개 이상이면 나란히 배치됩니다.
(현재: ① 개척이야기(상시) + ② 여름성경학교(6/1~8/15) 2개)

각 배너 항목:

| 항목 | 설명 |
|---|---|
| `active` | `true`=표시 / `false`=숨김 |
| `badge` | 노란 배지 글자 (예: "여름성경학교") |
| `title` / `description` | 제목과 한 줄 설명 |
| `link` | 누르면 이동할 곳 (보통 블로그 행사 글 주소) |
| `image` | 포스터 이미지 주소 (블로그 글의 카드 썸네일 주소를 그대로 써도 됨) |
| `start` / `end` | 표시 기간 — **end가 지나면 자동으로 사라짐** (내릴 걱정 불필요) |

새 행사 때는 이 파일의 내용만 갈아끼우면 끝. 행사 글을 먼저 블로그에 쓰고, 그 글 주소를 `link`에 넣는 방식을 권장합니다.

## 이웃사랑 섹션 관리법

홈페이지 하단에 이웃사랑(나눔·구제) 이야기가 고정 표시됩니다. **`site\data\outreach.json`**에서 관리:
- 현재: 성탄헌금 장애자녀가정 지원(2026-02) + 굿윌스토어 물품기증(2025-05)
- 새 나눔 이야기가 생기면: 블로그에 글 작성 → outreach.json에 항목 추가(제목/한줄소개/링크/날짜/이미지)
- "더 보기" 버튼은 블로그 이웃사랑 카테고리로 연결됨
